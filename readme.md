# Picture Puzzle

Randomly generates a sliding puzzle for the user to solve.


### Prerequisites

```
node
yarn
```

### Installing

```yarn install```

```
### Commands

#### Launch a local development server

```
yarn run dev
```

#### Generates a build
```
yarn run build
```

#### Run the tests
```
yarn test
```

Based on: https://github.com/photonstorm/phaser-examples/blob/master/examples/games/sliding%20puzzle.js
